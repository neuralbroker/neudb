# neuDB brand assets

Canonical logo files for README, PyPI, GitHub, and the website.

| File | Use |
|------|-----|
| `logo-mark.jpg` | README hero, social previews, marketing |
| `logo.svg` | Header wordmark (mark + neuDB) |
| `icon.svg` | Favicon, small UI placements |
| `app-icon.jpg` | App store / Apple touch icon |

## Design

- **Mark:** lowercase **n** + memory node + connection line
- **Colors:** `#000000` background, `#ffffff` foreground
- **Style:** minimal, xAI/Grok-inspired monochrome

## Website sync

The landing page copies these into `website/assets/`. After updating brand files here, run:

```bash
cp docs/brand/* website/assets/
```
